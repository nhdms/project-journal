---
name: pj-tree
description: Show ASCII tree of all phases and tasks with status icons
---

Run `pj tree` and present the output. The tree groups tasks under their phases, with an "Unassigned" bucket for tasks without a phase. Status icons: ✅ completed, 🚧 in_progress, ⬜ todo, ⚠️ partial, 🚫 blocked, 📝 needs_review.

!`pj tree`
